Bitboard-based representation for crate::chess::position::Position. Bitboards utilize the fact that modern processors operate on 64 bit integers, and the bit operations can be performed simultaneously. This results in very efficient calculation of possible attack vectors and other meaningful features that are required to calculate possible moves and evaluate position. The disadvantage is complexity that comes with bitboard implementation and inefficiency of some operations like â€œget piece type on given squareâ€ (efficiently handled by Square-centric board implementations that can be used together bitboard-based approach to compensate its shortcomings).

The implementation is based on PEXT Bitboards idea, which is an improvement over Fancy Magic Bitboards.

For visualizing and debugging the bitboards, there is a BitboardCalculator tool.
